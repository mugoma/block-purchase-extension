How to use Block Purchase Extension

How to use the Timer Function.

TL;DR: Start a timer by clicking on the extension icon, then click the big brown button. A timer will begin. Click the extension icon to see how much time is left. Leave feedback for yourself when the timer is done by clicking either the 'Yes' or 'No' buttons on the popup page.

To use this function, ensure the "Reflection(Timer)" option is turned on in the options page

There are two ways to start a timer: using the extension popup and through the 'Start Timer' button in the shopping website.

Starting the timer using the extension popup

The popup is the little window that appears when you click on an extension's icon on the browser.

The popup is normally accessible from all web pages. To open the pop-up click on the extension icon.(If it is not already pinned, click on the puzzle pice icon on the top right and click on the Block Purchase Extension Icon.)

To start a timer, click on the 'Yes! Add 24 hour countdown' button: a 24-hour countdown will start. You can check how much time is left by clicking on the popup window.

Starting the timer using in-page button.

If you are on a page that is supported, you'll see a 'Start Timer' button in the page.

Click on the button to start a 24-hour countdown

Check the remaining time.

To check how much time is left in the countdown, open the extension pop-up. There will be a countdown displayed on the page.

If the in-page button is present, it will show the time left.

Give feedback when countdown is complete.

After the timer is complete, you can record the action you took after. To do, open the extension popup. Read the question that is present and answer appropriately.

How to use the Hide Reviews Function.

TL;DR: Check the box in the options page. Visit a supported webpage(Amazon & Shien)

To use this function, ensure the "Social Influence(User Reviews)" option is turned on in the options page. The reviews section will be blurred, and the blur can be dismissed with a click.

Alternative tools

This section provides you with the information on other tools you can minimise the presence of dark patterns on the webpages you visit.

Disguised Ads

You can use an add-blocker to remove ads on the web-pages you visit. Ad-blockers such as Ad Guard and Ghostery work on most web-pages.
